# Optional Co-Creation Features

The routes under `/app` are **optional secondary features** for deeper synchronous interaction.

## Features

- **Mood Selection** (`/app`) - Select a mood for live matching
- **Matching** (`/app/matching`) - Wait for a partner with the same mood
- **Shared Moment** (`/app/moment/[id]`) - Complete "Finish My Thought" ritual together
- **Room** (`/app/room/[id]`) - Persistent space with chat and moment history

## Purpose

These features allow users to:
1. Match live with another user based on mood
2. Co-create moments through structured rituals
3. Build deeper connections in private rooms
4. Source emotional fragments for public spaces

## Access

- Requires authentication (Google OAuth)
- Accessible via `/app` routes
- Not required for core Emotional Platform experience

## Integration

Fragments created during co-creation moments can be shared to public Shared Emotional Spaces, enriching the emotional landscape.
